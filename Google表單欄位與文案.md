# Space Reset｜Google 表單欄位與文案

> 目的：讓客戶或合作夥伴快速上傳照片、平面圖與實際使用狀況，方便 Space Reset 找出優先處理位置。
> 原則：不要做成診斷問卷。只收「能不能開始檢測」所需資料。

## 表單基本設定

- 表單名稱：`Space Reset｜空間檢測資料上傳`
- 表單說明：

```text
請上傳你想請我們檢測的空間資料。

準備很簡單：手機拍的照片，加上平面圖或手繪圖，再寫下你最在意的實際狀況。我們會先確認資料是否足夠，再開始檢測。

這份表單不是醫療、心理、裝修設計或其他專業服務；它的目的，是協助我們了解空間現況，找出最值得優先處理的位置。
```

- 收集電子郵件：關閉
- 限制填寫一次：關閉
- 允許編輯回覆：關閉
- 回覆目的地：建立新的 Google Sheets
- 回覆試算表名稱：`Space Reset｜空間檢測回覆`
- 新回覆通知：開啟

## 第 8 題選項

題目：`你最想先處理的空間狀況`

說明：

```text
可複選。請依你的空間類型勾選最有感的項目；不需要全部都選。
```

選項：

- 【居家】回到家卻沒有真正放鬆
- 【居家】某些區域總讓人不想停留
- 【居家】整理後很快恢復原狀
- 【居家】家人容易疲累煩躁
- 【商業】客人進來不容易停留
- 【商業】入口、動線、收款區不順
- 【商業】員工容易疲累分心
- 【商業】調整過但現場狀況反覆發生
- 其他／說不上來，但覺得空間狀態不太對

## 第 11 題照片上傳說明

題目：`空間照片`

說明：

```text
請盡量包含：
1. 入口或大門
2. 主要活動區域
3. 走道或動線
4. 收納、櫃台、床位、座位或工作區
5. 你覺得最卡、最悶、最不舒服的位置

照片不用修圖，能看清楚現場即可。
```

如果用一鍵建立表單腳本，題目會是 `空間照片連結`。請填寫者先把照片放到 Google Drive、Dropbox、iCloud 或其他雲端資料夾，再貼上連結。

## 網站直送表單（主要流程：90 坪以下送出即跳轉付款）

網站 upload 區已內建表單，客戶填完按「送出資料，前往付款」會：

1. 資料經 Apps Script `doPost` 寫入回覆試算表的「網站直送」分頁
2. Telegram 立即通知（90 坪以下標示「網站直送（已跳轉付款頁）」；90 坪以上標示「網站直送（大型空間待評估）」）
3. 同步寄出含付款連結的 Email 備份（沒付成功可從信箱補付）
4. 瀏覽器直接跳轉綠界／藍新付款頁——全程不離開流程

例外：如果「大約坪數」填到 90 坪以上，網站按鈕會改成「送出資料，等候專案評估」。送出後不會跳付款頁，前端也會把空間類型送成含「先不確定」的文字；即使 Apps Script 還沒更新到最新版，也會走確認信，不會寄固定付款連結。由 Erick 先看資料後人工回覆檢測費與後續方案。

啟用步驟：

1. Apps Script 專案 → 部署 → 新增部署作業 → 類型選「網頁應用程式」
   - 執行身分：我
   - 誰可以存取：所有人
2. 複製部署網址，貼到網站 `script.js` 最上方的 `WEB_APP_URL`
3. 把綠界／藍新付款連結貼到 `script.js` 的 `PAYMENT_LINKS.home.url` 與 `PAYMENT_LINKS.business.url`
4. 每次修改 Apps Script 程式碼後，要「管理部署作業 → 編輯 → 新版本」才會生效
5. 到綠界／藍新後台，把「付款完成返回網址（Client Back URL / 返回商店）」設為 `https://你的網域/thanks.html`——客戶付完款會回到感謝頁，看到接下來 3 個工作天的流程，閉環才完整
6. 付款連結的「備註／留言」欄位：把欄位說明文字改成「請填寫你表單用的姓名與 Email，方便我們核對資料」，並設為必填。這是付款對回表單資料的核對鍵——綠界給你的付款人資訊不一定等於表單填寫人

備援：`WEB_APP_URL` 留空時，按鈕自動退回開啟 Google 表單的舊流程，網站不會壞。

## 自動付款信設定（Apps Script）

`apps-script-telegram-notify.js` 已包含自動付款信功能。啟用步驟：

1. 到綠界或藍新後台，建立兩個固定金額付款連結：居家 NT$2,500、商業 NT$3,500。
2. 在 Apps Script 的「專案設定 → 指令碼屬性」新增：
   - `PAYMENT_LINK_HOME`＝居家付款連結
   - `PAYMENT_LINK_BUSINESS`＝商業付款連結
3. 觸發條件維持原本的「表單提交時」即可，不用另外加。

寄送邏輯：

- 空間類型含「居家」→ 寄居家連結（NT$2,500）
- 其他類型（店面／工作室／辦公室／診所或美容院／其他商業空間）→ 寄商業連結（NT$3,500）
- 大約坪數超過 90 坪 → 寄確認信不含付款連結，由你人工評估後回覆
- 選「先不確定」→ 寄確認信不含連結，由你人工判斷後回覆
- 找不到 Email 或缺付款連結 → 不寄信，Telegram 通知會標示「請人工回覆」

每筆 Telegram 通知最後都會顯示付款信寄送狀態，漏接一眼就能看到。

## Google Forms 小提醒

Google 表單的「檔案上傳」通常會要求填寫者登入 Google 帳號，而且 Apps Script 無法自動建立檔案上傳題。  
第一版建議直接收雲端資料夾連結，讓填寫者不用卡在 Google 登入。

## 不建議放進第一版的欄位

- 生日、八字、命理資料
- 太多身心狀態題
- 30 天追蹤指標
- 價格方案選擇
- 付款資料
- 過度細的平面圖問卷

第一版只需要做到：有人上傳資料，我們收到通知，人工判斷下一步。
